# Countries App

### Technologies:
- Next.js 13
- TypeScript
- Tailwind CSS

### Demo:
https://next-js-project-countries-app.vercel.app

## Getting Started Locally
First, run the development server:

```Clone the project
# then
npm run dev
# or
yarn dev
# or
pnpm dev
```
Open http://localhost:3000 with your browser to see the result.
